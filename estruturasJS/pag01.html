Estruturas de Repetição em JavaScript

APRENDENDO SOBRE AS ESTRUTURAS DE REPETIÇÃO EM JS.

As estruturas de repetição são essenciais para automatizar tarefas no JavaScript, poa elas permitem executar um bloco de código várias vezes sem a necessidade de repetir o mesmo código manualmente. As principais estruturas de repetição em JavaScript são:

  • for
  • while
  • do...while

Essas estruturas ajudam a tornar o código mais eficiente, permitindo que tarefas repetitivas sejam feitas de maneira rápida e limpa. Vamos explorar cada uma delas.

Estrutura for

O for é usado quando sabemos exatamente quantas vezes o código precisa ser repetido. Sua sintaxe é:

for (inicialização; condição; incremento) {
    // código a ser executado
}

Exemplo 1: Listando números de 1 a 10

for (let i = 1; i <= 10; i++) {
    console.log(i);  // Exibe os números de 1 a 10
}

Exemplo 2: Calculando o fatorial de um número

let numero = 5;
let fatorial = 1;
for (let i = 1; i <= numero; i++) {
    fatorial *= i;
}
console.log(fatorial);  // Resultado: 120

Exemplo 3: Listando os números pares de 2 a 20

for (let i = 2; i <= 20; i += 2) {
    console.log(i);  // Exibe os números pares de 2 a 20
}

Exemplo 4: Contagem regressiva de 10 a 1

for (let i = 10; i >= 1; i--) {
    console.log(i);  // Exibe os números de 10 a 1
}

Exemplo 5: Listando números ímpares de 1 a 30

for (let i = 1; i <= 30; i += 2) {
    console.log(i);  // Exibe os números ímpares de 1 a 30
}

Estrutura while

O while é utilizado quando não sabemos o número exato de repetições, mas temos uma condição que deve ser satisfeita para continuar a execução. Sua sintaxe é:

while (condição) {
    // código a ser executado
}

Exemplo 1: Soma dos números até que o total seja maior que 20

let soma = 0;
let i = 1;
while (soma <= 20) {
    soma += i;
    i++;
}
console.log(soma);  // Exibe o valor da soma quando ela ultrapassa 20

Exemplo 2: Contar quantas vezes um número aparece em um array

let numeros = [1, 2, 3, 4, 2, 2, 5];
let contador = 0;
let i = 0;
while (i < numeros.length) {
    if (numeros[i] === 2) {
        contador++;
    }
    i++;
}
console.log(contador);  // Exibe 3, pois o número 2 aparece 3 vezes

Exemplo 3: Continuar pedindo a senha até que esteja correta

let senhaCorreta = "1234";
let senhaUsuario;
while (senhaUsuario !== senhaCorreta) {
    senhaUsuario = prompt("Digite a senha:");
}
console.log("Senha correta!");

Exemplo 4: Criando um loop para processar dados enquanto houver elementos em um array

let dados = [1, 2, 3, 4, 5];
let i = 0;
while (dados[i] !== undefined) {
    console.log(dados[i]);
    i++;
}

Exemplo 5: Criando uma fila de atendimento com espera condicional

let atendimento = ["Cliente 1", "Cliente 2", "Cliente 3"];
while (atendimento.length > 0) {
    let cliente = atendimento.shift();
    console.log(`Atendendo ${cliente}`);
}

Estrutura do...while

O do...while garante que o código será executado pelo menos uma vez, independentemente da condição inicial. Sua sintaxe é:

do {
    // código a ser executado
} while (condição);

Exemplo 1: Pedir um valor aleatório até que seja maior que 5

let numero;
do {
    numero = Math.floor(Math.random() * 10) + 1;
    console.log(numero);
} while (numero <= 5);

Exemplo 2: Continuar exibindo a mensagem de erro até que o usuário digite a palavra correta

let palavra;
do {
    palavra = prompt("Digite 'OK' para continuar:");
} while (palavra !== "OK");
console.log("Palavra correta!");

Exemplo 3: Menu de opções repetido até que o usuário escolha a opção de sair

let opcao;
do {
    opcao = prompt("Escolha uma opção: 1) Iniciar, 2) Sair");
} while (opcao !== "2");
console.log("Saindo...");

Exemplo 4: Pedir o nome do usuário até que seja fornecido um nome válido

let nome;
do {
    nome = prompt("Qual é o seu nome?");
} while (!nome);
console.log(`Olá, ${nome}!`);

Exemplo 5: Validar o formato de um número de telefone

let telefone;
do {
    telefone = prompt("Digite seu telefone (formato: (XX) XXXX-XXXX):");
} while (!/\(\d{2}\) \d{4}-\d{4}/.test(telefone));
console.log("Telefone válido!");
